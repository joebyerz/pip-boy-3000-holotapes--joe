iPip Joe Edition

A modified version of iPip created by Joe Byers.

This version was developed as a personal quality-of-life update to the original iPip Media Player and is intended to preserve the original experience while adding a number of usability improvements.

Highlights
Continuous sequential playback
Wraparound playlist playback
Marquee scrolling for long track titles
Improved wheel responsiveness
Encoder click-jump fix
Custom file order support
Increased filename/path limit
Improved readability of station names
Credits

Original iPip Media Player created by Cody Tolene.

This project is not intended to replace the official iPip release. It exists as an alternative version for users who would like to try these modifications.
