# iPip v1.0.3 (Joe Edition)

A modified version of iPip Media Player by Joe Byers.

This version was developed as a personal quality-of-life update to the original iPip Media Player and is intended to preserve the original experience while adding a number of usability improvements.

## Highlights

- Added continuous sequential playback (tracks now automatically advance to the next track)

- Added wraparound playback when the end of a playlist is reached

- Added protection against restarting a track that's already playing

- Added scrolling/marquee to now-playing text for long track titles (larger title, beneath waveform)

- Increased filename/path limit from 56 to 200 characters (character limit increase may not help, since the online uploader itself appears to enforce its own filename restrictions before files reach the Pip-Boy)

- Changed forced alphabetical sorting to sort by order of upload

- Increased default volume from 7 to 10

- Adjusted wheel responsiveness to more closely match the behavior of the stock Pip-Boy UI (Reduced knob scroll delay from 30ms to 7ms, reduced button debounce from 50ms to 10ms)

- Fixed the track selection cursor occasionally jumping up one space when clicking the wheel to select a track

- Added "SELECT" sound when stopping playback

- Added "SELECT" sound to "Random" button

- Slightly increased playlist/folder name size for better visibility


Original iPip Media Player created by Cody Tolene.

Many thanks to Cody for creating and sharing the original project with the Pip-Boy community.

This project is not intended to replace the official iPip release. It exists as an alternative version for users who would like to try these modifications.
